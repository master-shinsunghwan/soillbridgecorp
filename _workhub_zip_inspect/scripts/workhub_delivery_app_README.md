# 업무허브 발주 자동화 웹앱

`scripts/workhub_delivery_app.py`는 브라우저에서 엑셀을 업로드해 아래 두 가지 작업을 처리합니다.

- `택배건 요약`: 주소일브릿지 엑셀을 전달용 텍스트로 변환
- `송장번호 추출`: 출고송장 엑셀을 수하인명별 송장번호 엑셀로 변환
- `롯데택배 발주서 변환`: 주소일브릿지 엑셀을 롯데택배 발주서 양식에 맞춰 변환
- `차량인수증 생성`: 입력값을 차량인수증 양식에 맞춰 엑셀로 출력

## 실행

```powershell
python scripts\workhub_delivery_app.py 8765
```

브라우저에서 아래 주소를 엽니다.

```text
http://127.0.0.1:8765
```

## 송장번호 추출 규칙

출고송장 엑셀에서 아래 컬럼을 자동으로 찾습니다.

- 이름: `수하인명`, `받으시는분`, `받는분`, `수취인`, `고객명`, `이름`
- 송장번호: `운송장번호`, `송장번호`, `송장 번호`, `택배번호`

처리 방식:

- 원본 엑셀에서 처음 나온 이름 순서를 그대로 유지합니다.
- 같은 이름이 여러 번 나오면 한 줄로 합칩니다.
- 송장번호는 원본에 나온 순서대로 `/`로 이어 붙입니다.
- 합포장처럼 같은 이름에 같은 송장번호가 반복되어도 원본 건수대로 남깁니다.

결과 예시:

```text
수하인명              송장번호
양구군농협 예금계     258927933434 / 258927933445
이현화                258927933456
```

## 단독 실행

웹 화면 없이 송장번호 추출만 실행할 수도 있습니다.

```powershell
python scripts\invoice_number_exporter.py "C:\path\to\출고송장.xlsx"
```

결과 파일은 기본적으로 아래 폴더에 저장됩니다.

```text
output\invoice_numbers
```

롯데택배 발주서 변환만 실행할 수도 있습니다.

```powershell
python scripts\lotte_order_form_converter.py "C:\path\to\주소일브릿지.xlsx" "C:\path\to\롯데택배_양식.xlsx"
```

처리 방식:

- 원본 `판매사`를 양식 `주문번호`에 넣습니다.
- 같은 판매사가 여러 번 나오면 원본 순서대로 `키친쿡-1`, `키친쿡-2`처럼 번호를 붙입니다.
- 한 번만 나오는 판매사는 그대로 둡니다.
- 결과 파일명은 실행일 기준 `2026년 6월 13일`처럼 월/일을 채웁니다.

차량인수증은 웹 화면에서 직접 입력합니다.

입력 항목:

- 공급받는자
- 제품명 / 수량
- 납품장소
- 담당자명

처리 방식:

- 제품 입력칸은 기본 10줄을 표시합니다.
- 10개 이하 입력 시 출력도 10줄 기준으로 만듭니다.
- 제품 줄을 추가하면 입력한 제품 수만큼 출력 줄을 늘립니다.
- 출력 범위는 A4 한 장에 맞춰 설정합니다.
