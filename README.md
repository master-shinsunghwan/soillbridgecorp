# (주)소일브릿지 업무자동화 CRM

엑셀 기반 발주, 택배, CS, 수출입, 사내 업무관리를 한 화면에서 처리하기 위한 로컬 업무 포털입니다.  
회사 포털, 직원 중심 업무관리, 사내 메신저, 연차 관리, 프로젝트 캘린더를 함께 제공해서 작은 팀이 매일 보는 운영 현황판처럼 사용할 수 있습니다.

## 주요 기능

- 회사 포털: 공지사항, 사규/가이드, 직원 조직도, 사내 메신저, 캘린더
- 업무관리 CRM: 내 업무, 직원별 업무, 업무보드, 저장뷰, 댓글/활동 이력
- 프로젝트 추적기: 프로젝트별 진행률, 지연 업무, 오늘 마감 업무 확인
- 사내 메신저: 전체방과 직원 DM, `/업무 @직원 업무내용 / 기한` 명령으로 업무 생성
- 캘린더: 프로젝트 일정, 업무 마감, 연차/승인대기 연차 표시
- 연차 관리: 연차 신청, 승인/반려, 직원별 잔여 연차 관리
- 엑셀 자동화: 송장번호 추출, 롯데택배 발주서 변환, 차량인수증 생성
- 운영 도구: 백업/복원, 사용자 계정 승인, 권한 관리

## 실행 방법

### 1. 의존성 설치

```powershell
npm install
```

### 2. 앱 실행

기본 포트로 실행하려면:

```powershell
python _workhub_zip_inspect/scripts/workhub_delivery_app.py
```

8770 포트로 실행하려면:

```powershell
python _workhub_zip_inspect/scripts/workhub_delivery_app.py 8770
```

브라우저에서 접속:

```text
http://127.0.0.1:8770/
```

### 3. 화면 빌드 확인

```powershell
npm run build
```

## 기본 사용 흐름

1. 로그인 후 회사 포털에서 공지사항과 캘린더를 확인합니다.
2. 업무관리에서 `대시보드`, `내 업무`, `업무보드`, `직원` 탭을 확인합니다.
3. 사내 메신저에서 전체 공유나 직원 DM을 보냅니다.
4. 업무 지시는 `/업무 @직원 업무내용 / 기한` 형식으로 입력합니다.
5. 캘린더에서 프로젝트 일정, 업무 마감, 연차 일정을 한 달 단위로 확인합니다.
6. 관리자 계정은 사용자 승인, 권한, 백업/복원을 관리합니다.

## 메신저 업무 명령 예시

```text
/업무 @홍길동 공급사 견적서 확인 / 오늘 5시
```

성공하면 채팅창에 `TASK-000X 업무가 등록됐습니다` 형태로 결과가 표시되고, 업무보드와 캘린더에도 반영됩니다.

## FAQ

### Q. 계정 등록 후 바로 로그인할 수 있나요?

아니요. 계정 등록 요청 후 관리자가 승인해야 로그인할 수 있습니다.

### Q. 캘린더에는 어떤 일정이 보이나요?

프로젝트별 주요 마감, 업무 마감일, 승인된 연차, 권한이 있는 경우 승인대기 연차가 표시됩니다.

### Q. 연차 정보는 모든 직원에게 보이나요?

기본적으로 연차 조회 권한이 있어야 보입니다. 승인/관리 권한이 있는 사용자는 팀 연차와 승인대기 연차를 볼 수 있습니다.

### Q. 거래처가 없는 업무도 만들 수 있나요?

네. 직원 지시 업무처럼 거래처 없이도 저장됩니다.

### Q. 업무를 빠르게 찾으려면 어떻게 하나요?

업무보드의 검색과 필터를 사용하면 됩니다. 업무 번호, 제목, 설명, 담당자, 요청자, 거래처/직원 지시 업무 기준으로 찾을 수 있습니다.

### Q. 데이터를 GitHub에 올려도 되나요?

코드는 올려도 되지만 `.env`, DB 파일, `node_modules`, `dist`, 백업 파일, 생성된 실행파일은 올리면 안 됩니다.

## GitHub 제외 대상

다음 항목은 커밋하지 않습니다.

- `.env`
- DB 파일과 설정 비밀값
- `node_modules`
- `dist`
- 백업 파일
- 생성된 `exe`, `zip`, 업로드/다운로드 산출물
